Root Cause Analysis

“SOE of Control system or alarm logger lists the sequence of events, but does not help identifying the Root Cause” We need to check with other process parameters as well to conclude.
“Process alarms & parameters alone may not be enough. Many times we need to use our experience in determining Root Cause. We have set up an investigation committee it may even take weeks”. SAMA RCA is solution to the above issues raised frequently in Process Industries, Power plants as well as Building automation.
SAMA RCA is online software with instant Root Cause analysis solution. Root Cause Analysis (RCA) package gives instant Root Cause of not only a shutdown, but also of an upset condition.
RCA over view display lists all possible upsets ot trip conditions or Decision Tree names and their status. If an upset condition results, then status will turn red. From here, user can go to Root cause view to view the Root Cause; even if it is green user may to go Normal view to see the current status of the plant and may take immediate steps to prevent an upset.

Root Cause Identification:

Decision Tree mechanism based on Result Tag (trip or upset), Condition tags and Cause tags as well as statistical calculations. Statistical calculations supplement Decision Tree, especially in cases such as: - Cause tag or condition tag sensor fails - More than one Cause tag - Hidden causes, hardware failure

Most Probable Root Cauase:

If auto mode is chosen, auto relevance calculated (self‐ learnt automatically or exploratory) is used to calculate adjusted probability.
