SAMA Data Collector

Accurate and highly reliable collection of data and alarms in the right format is the first step for performing any form of analysis. Introducing, the brand new SAMA Data Collector that connects with any data source or platform and provides output data in the format required by external clients or SAMA Tunneller.
  • Data sources include OPC(UA/DA/HDA/AE), ODBC or Modbus TCP/IP
  • Data output formats include OPC(UA/DA/HDA/AE), ODBC, Text File or SAMA tunneller
  • Multiple output clients of specific format
  • Handling of source redundancy
  • Handling of historical data

    SAMA RT Historian

    Data analysis involves performing data mining and data manipulation in the form of calculations over a specific time interval or applying a dead band to eliminate useless data. SAMA RT Historian connects to SAMA Data Collector and performs these and many more functionalities.
    • Data sources includes SAMA Data Collector
    • Data output formats include OPC (UA/DA/HDA) or SAMA Tunneller
    • Multiple output clients of specific format
    • Configuration and handling of calculated data
    • Configuration of dead band
    • Configuration and handling of synchronous or asynchronous data
